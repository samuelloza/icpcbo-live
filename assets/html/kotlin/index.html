Kotlin local

Referencia rápida para compilar y ejecutar Kotlin en esta máquina.

Comandos

  • Compilar jar: kotlinc Main.kt -include-runtime -d main.jar
  • Ejecutar jar: java -jar main.jar
  • Compilar estilo ICPC: kotlinc Main.kt -d .
  • Ejecutar clase: kotlin -classpath . MainKt

Editor recomendado

IntelliJ IDEA Community.
